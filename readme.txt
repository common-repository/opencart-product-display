== Description ==

OpenCart Product Display is a WordPress plugin that allows you to show products
from your separate OpenCart store in your WordPress site.

== Installation ==

1. Upload the the plugin folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your store database details in the 'Settings > OpenCart Product Display' window.
4. Add the 'OpenCart Product Display' widget to one of your sidebars.

== Frequently Asked Questions ==

= Will this work for any online store or only OpenCart stores? =

No, this plugin was built to only work with OpenCart stores.

= What options can I change from the widget settings? =

You can give the widget a title and select the number of products to generate. More options to be added shortly.
